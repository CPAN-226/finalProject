Title: Chat Application in a Virtual Environment.
Description: Develop a chat application that allows communication between multiple
users running on separate virtual machines (VMs) within a VirtualBox environment. The
goal is to simulate inter-machine communication by setting up a network between two
or more VMs and deploying a Python-based chat service on each. The application can be
developed using Python with Django or similar tools. Following are some core features:
• Install a Virtual box on your laptop/pc (host machine).
• On the virtual box install minimum two windows virtual machines (VMs).
• Set up a network between them.
• Ensure both VMs are on the same network and can ping each other.
• Now write a chat application to communicate among the VMs.
• Each user should have a dedicated chat window/interface or page for interaction.
• The interface should clearly display the username.
• The chat interface should dynamically update with the most recent messages.
• All the conversation should be stored in a file or database in chronological order.
